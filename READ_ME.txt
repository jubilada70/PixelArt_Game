e tem inspirações em EarthBound,Stardew Valley,Undertale.

//Funcionando
Time of Day,Movimentação do Personagem, Ciclo do Dia.

//Não Funcionando
Botão de startar o diálogo.
 
//Tilesets
Os tilesets ainda não foram criados, foi criado apenas uns testes 
com tilemaps separados.

//Imagens 
As Pixel Arts estão sendo feitas utilizando o Krita e o Piskel.


//Animações
As animações foram feitas com o Bend Tree da Unity,

W,A,S,D - movimentam o personagem.

This is a rpg game made in pixel art, the game presents a cool idea
and has inspirations in EarthBound, Stardew Valley, Undertale.

//Working
Time of Day, Character Movement, Cycle of the Day.

// not working
Start dialogue button.
 
// Tilesets
The tilesets have not yet been created, only a few tests have been created
with separate tilemaps.

//Images
Pixel Arts are being made using Krita and Piskel.


// Animations
The animations were made with Unity's Bend Tree,
however, they are not yet complete.

// Commands
W, A, S, D - move the character.
Mouse Scroll - Zoom in and out.
